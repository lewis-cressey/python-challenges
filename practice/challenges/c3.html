Ask the user to enter their age.

Print "happy birthday" once for every year old they are.

Convert the input to an integer using int().

Use a for loop with range() to repeat the print statement.

import random def generate(i): age = random.randint(1, 10) i.write(str(age)) def validate(i, o): age = int(i.read()) for i in range(age): assert o.read() == "HAPPY BIRTHDAY"
