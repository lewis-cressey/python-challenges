Print all palindromic numbers from 11 to 999 inclusive.

A palindromic number reads the same forwards and backwards (e.g., 121).

Print each number on a new line. The order does not matter.

Use a loop to iterate through numbers from 11 to 999.

Convert each number to a string to check if it equals its reverse.
