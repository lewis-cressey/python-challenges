Ask the user to enter a word.

Count how many times the letter 'a' (upper or lower case) appears in the word.

Print the count as a number.

Use input() to get the word.

You can use the string method .upper() to change text to UPPER CASE.

You can loop through the letters in text using a for loop like this: for letter in text: print(letter).

import random def generate(i): words = ["Apple", "Banana", "Car", "Door", "Elephant", "Fish", "Grape"] word = random.choice(words) i.write(word) def validate(i, o): word = i.read() expected = str(word.count('A')) assert o.read() == expected
