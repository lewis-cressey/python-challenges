Ask the user for the width and height of a rectangle.

Calculate the area and print only the resulting number.

Use input() to get values from the user.

Convert the input strings to integers using int().

Multiply width by height to get the area.

import random def generate(i): width = random.randint(1, 10) height = random.randint(1, 10) i.write(str(width)) i.write(str(height)) def validate(i, o): width = int(i.read()) height = int(i.read()) expected = str(width * height) assert o.read() == expected
